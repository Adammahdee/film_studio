# Film Studio Inventory System

A simple PHP-based inventory and request management system.

## Features

- User authentication
- Inventory management
- Request system
- Approval workflow

## Tech Stack

- PHP
- MySQL
- PDO

## Setup

1. Clone repo
2. Copy `config/db.example.php` to `config/db.php`
3. Configure database credentials in `config/db.php`
4. Import `database_SQL`
5. Run on Laragon or XAMPP
